## チャージ電流の設定方法

`R1` の値でチャージ電流が決まります：

```
I_charge [mA] = 1000 / R1 [kΩ]
```

| R1 | チャージ電流 | バッテリ(250mAh)に対するCレート |
|----|------------|-------------------------------|
| 4kΩ | 250mA | 1C（推奨、安全） |
| 2kΩ | 500mA | 2C（やや速い） |
| 1kΩ | 1000mA | 4C（使用しないこと） |

**推奨: R1 = 4kΩ（250mAh バッテリには1Cチャージが安全）**

## 回路メモ

### USB-C CC抵抗（R2, R3）
- CC1, CC2 それぞれ 5.1kΩ で GND へ
- これにより USB-C ホスト側が「このデバイスは5V/900mAを要求している」と認識

### nRF9161 電源
- VDDMAIN (×4) と VDDIO (×2) すべて VBAT へ接続
- 各ピンに個別 100nF デカップリングキャップ（C1〜C6）

### SIM インターフェース (nRF9161)
```
nRF9161       Nano SIM
P0.06 ──────→ VCC     (SIM 3V 電源出力)
P0.07 ──────↔ I/O     (双方向データ)
P0.08 ──────→ CLK     (クロック)
P0.09 ──────→ RST     (リセット)
P0.10 ←────── CD      (カード検出、PU 内蔵)
```

### SWDデバッグ (J6 ← J-Link)
```
J6 Pin    信号
1         VCC (3.7V)
2         SWDIO ↔ P0.18
3         SWDCLK → P0.17
4         GND
```
